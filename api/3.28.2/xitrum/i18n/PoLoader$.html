  10.  def get(language: String): scaposer.I18n

    For the specified language, this method loads and merges all i18n/<language>.po files in classpath.

    For the specified language, this method loads and merges all i18n/<language>.po files in classpath.

    You can store the po files in JAR files, at src/main/resources/i18n/<language>.po (compile time), or at config/i18n/<language>.po ("config" directory is put to classpath by Xitrum at run time, this is convenient if you want to modify po files at run time without having to recompile or restart).

    The result is stored in cache for further fast access. If you want to reload the po files, use clear() or remove(language) to clean the cache, then load again.

    returns

    Empty Po if there's no po file
